# NexaWave — Dynamic Website

A professional dynamic website built with Node.js + Express + EJS.
Deployed on Microsoft Azure App Service.

## Tech Stack
- **Backend**: Node.js + Express.js
- **Template Engine**: EJS
- **Frontend**: HTML5, CSS3, Vanilla JS
- **Hosting**: Microsoft Azure App Service

## Pages
- `/` — Home (Hero, Stats, Services, Team, CTA)
- `/about` — About page with team
- `/services` — Full services listing
- `/contact` — Contact form (POST handler included)
- `/api/stats` — JSON API endpoint

## Run Locally

```bash
npm install
npm start
```

Open http://localhost:3000
